# Тестовое задание для tzSaaSoft
Форма управления учетными записями

#№ Стек:
- Vue.js 3 + Composition API (фреймворк)
- TypeScript (язык)
- Pinia (стейт менеджер)
- UI фреймворк - element-plus

## Настройка проекта

```sh
npm install
```

### Запуск проекта в режиме разработки

```sh
npm run dev
```
